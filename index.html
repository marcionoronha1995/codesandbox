Variáveis e estruturas condicionais

Os nomes das variáveis

Regras

  • letras, dígitos ou símbolos $ e _
  • o primeiro carctere não pode ser um dígito
  • não pode ser uma palavra reservada

Tipos de dados

  • String
  • Number
  • Boolean
  • Object
  • Null
  • Undefined

Tipos de variáveis

  • VAR
    • Valor inicial PODE ser alterado
    • É possível criar outra váriavel com o mesmo nome
  • LET
    • Valor inicial PODE ser alterado
    • Não É possível criar outra váriavel com o mesmo nome
  • CONST
    • Valor inicial NÃO PODE ser alterado
    • Um nome de variável único
    • É obrigatório inicializar o valor da variável

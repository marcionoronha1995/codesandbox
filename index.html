Variáveis e estruturas condicionais

Os nomes das variáveis

Regras

  • letras, dígitos ou símbolos $ e _
  • o primeiro carctere não pode ser um dígito
  • não pode ser uma palavra reservada

Tipos de dados

  • String
  • Number
  • Boolean
  • Object
  • Null
  • Undefined

Tipos de variáveis

  • VAR
    • Valor inicial PODE ser alterado
    • É possível criar outra váriavel com o mesmo nome
  • LET
    • Valor inicial PODE ser alterado
    • Não É possível criar outra váriavel com o mesmo nome
  • CONST
    • Valor inicial NÃO PODE ser alterado
    • Um nome de variável único
    • É obrigatório inicializar o valor da variável

Tipos de operadores de comparação

Operador Operação Exemplo
== igual a “1” == 1 // true
=== Estritamente igual a “1” === 1 // false
>, < Maior que, menor que 20 > 20 // false
>=, <= Maior ou igual, menor ou igual 20 <= 20 // true
!=, !== Diferente de 1 != 2 // true

o JavaScript trata os valores de string vazia, o number 0, undefined, false e null como valores falsos (falsy).

Todos os demais valores são tratados como valores verdadeiros (truthy).

Estruturas condicionais

Múltiplas condicionais if ... else podem ser aninhados quando necessário. Observe que não existe elseif (em uma palavra).

O correto é a instrução com espaços (else if), conforme abaixo:

var idade = 20;
if (idade === 18) {
console.log('Acabou de ser maior idade');
} else if (idade > 18) {
console.log('É maior de idade');
} else {
console.log('É menor de idade');
}
